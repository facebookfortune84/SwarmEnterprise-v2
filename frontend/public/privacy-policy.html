8. Right to Erasure (Account Deletion)

You can permanently delete your account and all associated personal data by making an authenticated API request:

⚠ This action is irreversible Deleting your account will permanently erase your user record, all API keys, and all usage events linked to your account. Stripe billing records are retained for 7 years as required by US tax law and cannot be erased early. 
DELETE /api/user/account
Authorization: Bearer <your_access_token>

→ HTTP 204 No Content

Alternatively, email robertdemottojr50@gmail.com with the subject line "Account Deletion Request" and we will process the request within 30 days.

9. Data Portability & Export

You may download a complete JSON export of all personal data we hold about you — including your user profile, API keys, and usage events: 

GET /api/user/export
Authorization: Bearer <your_access_token>

→ HTTP 200 OK
Content-Type: application/json

The response is structured JSON you can save locally or import into another service. The exported file does not include your password hash.

12. Security

We implement the following technical and organisational measures:

  • Passwords stored exclusively as bcrypt hashes — plaintext is never persisted.
  • JWT-based authentication with short-lived access tokens (15 minutes) and Redis-backed revocation.
  • API keys stored and validated as opaque tokens; access scoped with read/write permissions.
  • All data in transit protected by TLS 1.2+.
  • Role-based access control enforced at the API layer.
  • Rate limiting per IP address to prevent brute-force attacks.

No method of transmission over the internet or electronic storage is 100% secure. We will notify affected users within 72 hours of becoming aware of a data breach that is likely to result in a high risk to their rights and freedoms.
